# Архитектура сети

## Задание:
  1. Разработать и задокументировать адресное пространство для лабораторного стенда
  2. Настроить IP адреса на каждом активном порту
  3. Настроить каждый VPC в каждом офисе и определить ему свой VLAN
  4. Настроить VLAN/Loopback интерфейсы для управления сетевыми устройствами
  5. Настроить сети офисов так, чтобы не возникало broadcast штормов, а использование линков было максимально оптимизировано
